# Hudson Tien Capstone Site

Static portfolio website for Hudson Tien's PLTW Engineering Design and Development capstone project.

## Files

- `index.html` - Portfolio website homepage.
- `slides-air-pressure-cannon.html` - Browser-based slideshow presentation.
- `air-pressure-cannon-presentation.pptx` - Editable PowerPoint fallback deck.
- `poster-air-pressure-cannon.html` - Browser-based poster draft.
- `assets/` - Compressed prototype photos and the original directions PDF.
- `.nojekyll` - Tells GitHub Pages to serve the files directly.

## Publish With GitHub Pages

1. Create a new GitHub repository named `hudson-capstone-site`.
2. Upload the files from this folder into the repository root.
3. Open the repository settings.
4. Go to `Pages`.
5. Under `Build and deployment`, choose `Deploy from a branch`.
6. Choose branch `main` and folder `/root`.
7. Save.
8. Wait for GitHub to show the published URL.

The final URL will look similar to:

`https://YOUR-GITHUB-USERNAME.github.io/hudson-capstone-site/`
